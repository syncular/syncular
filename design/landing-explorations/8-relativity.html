The protocol

Two cores.
One physics.

One written protocol is the law. Two conformance-locked cores obey it — TypeScript for the web, Rust for native via C FFI — verified against 74 shared scenarios so every platform experiences the same universe: Web, React, Swift, Kotlin, Flutter, React Native, Tauri, Rust.

Scopes as code

Authorization is a function you write once. The server validates every commit against it before anything crosses the horizon.

Offline outbox

Drift out of contact and keep writing. Local commits queue in the outbox and replay, in order, when you re-enter range.

Honest conflicts

Divergent histories surface as conflicts you resolve deliberately — never silently merged behind your back.

Realtime by default

Committed writes propagate to every subscribed client over WebSocket, straight off the accretion of the commit log.

CRDT columns

Opt individual columns into merge semantics when last-write-wins isn't the answer. Everything else stays plain SQL.

Blobs, E2EE, windows

Binary sync, optional end-to-end encryption, and windowed sync so clients pull only the slice of spacetime they need.

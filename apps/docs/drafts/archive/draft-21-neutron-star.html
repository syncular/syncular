Compressed feature set — everything, tightly packed
Feature Description Layer
Offline-first All reads/writes hit local SQLite. Network is optional, never blocking. client
Commit-log sync Append-only operations log. Pull deltas, push outbox. Conflict-free by design. core
Realtime WebSocket wake-ups trigger instant pulls. Sub-second propagation across clients. transport
Presence Ephemeral cursor/status sharing over the same WebSocket channel. transport
Scope keys Partition data per-user, per-team, per-anything. Fine-grained access boundaries. server
Snapshot chunks Bootstrap new clients from compressed snapshots instead of replaying full history. server
E2E encryption Client-side plugin encrypts payloads before they leave the device. Server is zero-knowledge. plugin
Blob storage Binary attachments synced alongside structured data. Lazy-loaded, deduplicated. core
Edge relay Deploy read replicas at the edge. Clients connect to nearest relay, relay syncs to origin. infra
React bindings Hooks for queries, mutations, sync status, presence. Suspense-compatible. client
Admin console Built-in web UI for inspecting tables, operations, connected clients, and sync health. server
Pruning Automatic compaction of old operations. Keep the log dense, never bloated. server

Capabilities

Every feature flows with purpose

Offline-First Engine

Write locally, sync when connected. Every mutation is queued in an outbox and replayed in order when the connection is restored.

Real-Time Sync

WebSocket-driven wake-ups push changes instantly. No polling. Presence awareness lets you see who is online and where.

Commit-Log Protocol

Every change is an immutable commit. Pull only the delta since your last sync point. Efficient, auditable, conflict-aware.

Scoped Subscriptions

Scope keys let each client subscribe to exactly the data it needs. The server sends only matching commits. Zero waste.

End-to-End Encryption

Optional E2EE plugin encrypts data on the client before it ever leaves the device. The server sees only ciphertext.

Blob Storage

Sync binary files alongside structured data. Blobs are chunked, deduplicated, and streamed efficiently to every client.

Get Started

Start flowing in seconds

Add Syncular to your project. The server runs on any Bun or Node runtime. The client works in browsers, React Native, and Electron.

$ bun add @syncular/client @syncular/server
$ bun add @syncular/transport-http @syncular/transport-ws
// Quick start
import { createClient } from '@syncular/client'
import { createHttpTransport } from '@syncular/transport-http'

const client = createClient({
  transport: createHttpTransport({
    baseUrl: 'https://api.example.com'
  })
})

syncular.init()

An offline-first sync engine. Write to local SQLite. Push commits to server. Pull merged truth back. Everything past the event horizon syncs.

syncular — event horizon
// 1. Define your sync handlers
const server = createServer({
  tables: {
    tasks: defineTable({
      columns: { title: string, done: boolean },
      scopes: (row) => [`user:${row.userId}`]
    })
  }
})
 
// 2. Client writes locally, syncs automatically
const client = createClient({ db: 'sqlite' })
await client.mutate('tasks', { title: 'Ship v1.0' })
 
// 3. Query at the speed of light
const tasks = await client.query('tasks')
// → [{title: 'Ship v1.0', done: false}] <0.1ms

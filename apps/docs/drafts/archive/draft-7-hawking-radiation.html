Syncular is an offline-first sync engine modeled after the event horizon. Local mutations are pushed inward. Server truth radiates outward to every client. Nothing is lost.

The mechanics of sync

Push — Matter falls in
→

Client writes to local SQLite — instant, offline

→

Mutations become ordered commits in the log

→

Commits pushed to server when connected

Pull — Truth radiates out
←

Server merges commits from all clients

←

Resolved state radiates to every subscriber

←

Clients converge — conflict-aware, scope-filtered

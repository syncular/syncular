# Features

Feature guides describe product capabilities once. Syncular's semantics are the
same across JavaScript and Rust clients: typed local reads, generated
mutations, scoped sync, realtime events, blobs, CRDT fields, encryption, and
recovery all come from the same Rust runtime foundation.

When a feature page includes code, the snippet is labeled by host surface. A
TypeScript snippet usually means one of two things: app-contract/server
configuration, or the browser/React binding. It is not the default client model
for every app. Use [JavaScript](/docs/javascript) or [Rust](/docs/rust) for
language-specific setup and API details.

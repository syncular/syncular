# Start

Syncular is a sync framework for building **local-first applications** — apps
that store data in local SQLite, work offline, and sync through a Rust-owned
client runtime when online.

> **Alpha.** APIs, wire formats, and storage layouts will change between releases. Pin versions and expect manual migrations between versions.

Sync systems deserve skepticism. This section is ordered so you can decide
whether Syncular fits, see it working end to end, and then choose how to adopt
it.

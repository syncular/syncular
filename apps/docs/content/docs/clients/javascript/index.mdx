# JavaScript

Use this section when your app code is TypeScript or JavaScript. The current
JavaScript client shape keeps app reads in typed Kysely, routes writes through
generated Syncular mutations, and delegates SQLite/sync internals to the
Rust-owned runtime where the target supports it.

## What JavaScript owns

- Kysely query building for application reads.
- Generated mutation helpers for application writes.
- React hooks and host bridge adapters.
- TypeScript/server/browser test fixtures through `@syncular/testkit`.

## What the runtime owns

- SQLite storage and migrations.
- Outbox, conflicts, row/field change events, blobs, CRDT fields, encryption,
  realtime wakeups, and retry behavior.
- Native app lifecycle when JavaScript runs in React Native, Tauri, or Electron
  renderer code.

For Rust application code and native SDK details, use [Rust](/clients/rust).

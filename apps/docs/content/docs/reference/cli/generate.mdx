# Generate Command

`syncular generate` is the app-facing codegen entry point from the `syncular`
npm package.

It runs the current generation pipeline in order:

1. If `syncular.app.ts` exists, or `--app` is supplied, generate
   `generated/syncular.codegen.json` from the app definition.
2. If there is no app definition and no codegen config, initialize a starter
   `generated/syncular.codegen.json` from migrations.
3. Run the Rust generator against the manifest directory.

The CLI resolves the Rust `syncular-codegen` binary from, in order:

1. `SYNCULAR_CODEGEN_BIN`, when set.
2. The repo-local Rust crate while developing Syncular itself.
3. Syncular's local tool cache.
4. `PATH`.
5. A Cargo install into the local tool cache, when Cargo is available.

## Usage

Generate files:

```bash
npx syncular generate --manifest-dir .
```

Check generated files in CI:

```bash
npx syncular generate --manifest-dir . --check
```

Use an explicit app definition:

```bash
npx syncular generate --manifest-dir . --app ./syncular.app.ts
```

Pre-install the Rust generator:

```bash
npx syncular codegen install
```

Pin or redirect the installed generator:

```bash
npx syncular codegen install --version 0.1.0 --root ./.syncular/codegen
SYNCULAR_CODEGEN_BIN=./.syncular/codegen/bin/syncular-codegen npx syncular generate --manifest-dir .
```

## Flags

- `--manifest-dir <path>`: project directory containing migrations and generated handoff config
- `--app <path>`: app definition used to generate codegen config
- `--migrations-dir <path>`: override the migrations directory passed to Rust codegen
- `--rust-output-dir <path>`: override the Rust generated output directory
- `--check`: fail if generated output is out of date

## Environment

- `SYNCULAR_CODEGEN_BIN`: explicit `syncular-codegen` binary path or command
- `SYNCULAR_CODEGEN_VERSION`: crate version to install when the package version is not the desired codegen version
- `SYNCULAR_CODEGEN_AUTO_INSTALL=0`: disable Cargo auto-install during `syncular generate`
- `SYNCULAR_CACHE_DIR`: override the default Syncular tool cache directory

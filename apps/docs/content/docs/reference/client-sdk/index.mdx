# Client SDK Reference

`@syncular/client` is the JavaScript package for the Rust-owned browser
worker runtime, Kysely-backed local database, and generated TypeScript
integration points. Rust app code uses the crates documented in
[Rust](/docs/rust). React apps use the separate `@syncular/react` package.

Use the language guides for the current package surface:

- [JavaScript quick start](/docs/javascript/quick-start)
- [Browser client](/docs/javascript/browser)
- [Rust client quick start](/docs/rust/quick-start)
- [Rust native runtime](/docs/rust/native-runtime)
- [Testing](/docs/testing)

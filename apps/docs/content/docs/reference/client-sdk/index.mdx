# Client SDK Reference

`@syncular/client` is the Rust-owned client package. It provides the browser
WASM/worker runtime, Kysely-backed local database, and generated TypeScript
integration points. React apps use the separate `@syncular/react` package.

Use the Rust client guides for the current package surface:

- [Rust client quick start](/docs/client/quick-start)
- [Browser client](/docs/client/browser)
- [Native client](/docs/client/native)
- [Testing](/docs/client/testing)

# Server

The server is the authoritative sync surface. It owns authentication, table
handlers, scope resolution, snapshot/pull behavior, push/apply behavior,
realtime wakeups, blob routes, audit APIs, and deployment-specific adapters.

Use this section for implementation guides. Exact API signatures and generated
HTTP endpoint details stay in [Reference](/reference).

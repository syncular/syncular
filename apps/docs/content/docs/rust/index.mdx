Rust is the canonical Syncular client runtime. Use this section when your app
code is Rust, or when a native shell embeds Syncular through the Rust runtime.

## What Rust owns

- SQLite storage and migrations.
- Diesel query-builder reads for Rust app code.
- Generated mutation namespaces for writes.
- Native worker queues, event streams, realtime, presence, conflicts, blobs,
  CRDT fields, encryption, and retry behavior.
- Native runtime artifact packaging for app hosts.

For browser, React, React Native, Tauri, Electron, and TypeScript testing, use
[JavaScript](/docs/javascript).
